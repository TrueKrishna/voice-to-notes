How API Key Rotation Works

  • Add multiple API keys from different Google accounts
  • When one key hits its quota limit, the system automatically switches to the next available key
  • Exhausted keys are marked in red - reset them when their quota refreshes (usually daily)
  • You can temporarily disable keys without deleting them
